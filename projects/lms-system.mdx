## 개요

학습 관리 기능과 사용자 권한 흐름을 구현한 웹 시스템 프로젝트입니다.

## 주요 작업

- 사용자 역할별 권한 처리
- 학습 콘텐츠 관리 흐름 구현
- MySQL 기반 데이터 모델링

## 회고

기능 구현뿐 아니라 권한 흐름과 데이터 구조를 먼저 정리하는 것이 유지보수성에 중요하다는 점을 경험했습니다.
