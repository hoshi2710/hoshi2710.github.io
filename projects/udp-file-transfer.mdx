## 개요

UDP 브로드캐스트를 활용한 파일 전송 프로그램의 안정성과 성능을 개선한 프로젝트입니다.

## 주요 작업

- UDP 기반 파일 전송 흐름 구현
- 패킷 손실 상황을 고려한 재전송 처리
- WinSock 기반 네트워크 프로그래밍 실험

## 회고

UDP는 가볍지만 신뢰성을 직접 설계해야 하므로, 전송 상태 관리와 예외 처리가 중요했습니다.
