# 설명
문제자체는 어렵지 않은데 오히려 스토리만 대강 훑고 지나가는 바람에 문제를 이해하는데 시간이 걸렸다. 핵심만 보면 매우 쉬운 문제였다.그냥 첫번쨰 자리에는 1번이, 두번째 자리는 2번이.. 이런씩으로 순서대로 오도록 하면 된다는 것이다. - 항상 문제 꼼꼼히 읽기
# 코드
```c
#include <stdio.h>

int main() {
    int n;
    scanf("%d",&n);
    
    int count=0;
    for(int i=0; i<n; i++)
    {
        int temp;
        scanf("%d",&temp);
        if(temp != i+1)
            count++;
    }
    printf("%d\n",count);
    return 0;
}

```
