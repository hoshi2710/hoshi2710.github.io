# 설명
그냥 각 케이스별로 총합구하는 문제.
더이상의 설명은 생략한다.

# 코드

```c
#include <stdio.h>

int main() {
    int n;
    scanf("%d",&n);
    for (int i=0; i<n; i++)
    {
        int case_amount, result=0;
        scanf("%d",&case_amount);
        for(int j=0; j<case_amount; j++)
        {
            int temp;
            scanf("%d",&temp);
            result += temp;
        }
        printf("%d\n",result);
    }
    return 0;
}

```
