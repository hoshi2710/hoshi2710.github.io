# 문제
![](/images/posts/2022-01-18-2869/2869.png)
[https://www.acmicpc.net/problem/2869](https://www.acmicpc.net/problem/2869)
# 설명
원래는 for문으로 몇분도 안되서 풀었으나 시간제한이 빡빡하게 걸려 있어서 어쩔수 없이 반복문을 포기하고 단순 사칙연산과 조건문만을 이용해서 구현하려다보니 시간이 조금 오래 걸렸다.<br>먼저 사용자로부터 세개의 값 을 a,b,v 변수에 넣고 day라는 변수를 따로 만들어서 ((v-a) / (a - b))+1 를 넣어 목적지 근처까지 가는데 걸리는 일수를 계산하여 넣고 저 일수동안 이동했는데도 애매하게 부족해서 더 가야하는 경우라면 day++문을 통해 날 수를 하루 더추가하여 최종적으로 일수를 출력하도록 만들었다. (낮에 해놓고 바로 올린다는게 깜빡해서 이제 올림)
# 코드
```c
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>
int main()
{
	int a, b, v,day;
	scanf("%d %d %d",&a,&b,&v);
	day = ((v-a) / (a - b))+1;
	if (a+((day-1)*(a-b))<v)
		day++;
	printf("%d", day);

	return 0;
}
```
